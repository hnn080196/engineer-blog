## Key Specifications

**Fonts:** Crimson Pro (headings) + Lora (body) + Fira Code (code)
**Colors:** #1a1a1a/#ffffff (light) | #ffffff/#0d1117 (dark)
**Accent:** #0066ff (light) | #4da6ff (dark)
**Spacing:** 4, 8, 16, 24, 32, 48, 64px scale
**Max-width:** 65 characters (~900px)
**Code theme:** Dracula (dark) / Atom One Light (light)
