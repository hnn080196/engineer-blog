# Exemplary Tech Blog Designs

## 1. swyx.io
**Layout:** Full-width hero + centered content grid (800px max)
**Colors:** Light bg (#fff), dark text (#222), accent: vivid teal/cyan
**Typography:** Inter/Poppins (headers), system fonts (body)
**Features:**
- Clean navigation bar with dark mode toggle
- Prominent author bio + social links
- Article cards with reading time + date metadata
- Syntax highlighting with code blocks
- Newsletter CTA prominently positioned

## 2. overreacted.io (Dan Abramov)
**Layout:** Centered single-column (~600px), asymmetric whitespace
**Colors:** Off-white background, black text, minimal accent usage
**Typography:** Georgia serif (headers), monospace for code, clean body copy
**Features:**
- Distraction-free reading experience
- Inline code examples with high readability
- Subtle shadows on interactive elements
- Footer with archive/about links
- Strong visual hierarchy through spacing

## 3. joshwcomeau.com
**Layout:** CSS Grid-based responsive, centered hero with featured articles below
**Colors:** Dark mode default (#1a202c), pastel accents (purple/blue), excellent contrast
**Typography:** Custom web fonts, large headlines, generous line spacing
**Features:**
- Interactive explorations (Viz.js components embedded)
- Newsletter signup with preview
- Article metadata (date, category, read time)
- "Estimated Time to Read" badge
- Animated transitions on scroll
- Deep content with visual explanations

## 4. tkdodo.eu (TKDodo - React Query Expert)
**Layout:** Minimal sidebar nav + main content area, responsive stacking
**Colors:** Light gray bg (#f5f5f5), dark navy text, blue accent links
**Typography:** System font stack (Performance), consistent 16px base
**Features:**
- Clear article hierarchy with TOC generation
- Code snippets with line highlighting
- Dark mode toggle in header
- Author bio at article end
- Related articles suggestions
- Clean, brutalist aesthetic

## 5. www.peterbe.com
**Layout:** Classic blog layout, sidebar for archives/categories
**Colors:** Minimal palette - white, gray, black with blue accents
**Typography:** Clean sans-serif, readable body copy (18-20px)
**Features:**
- Content-first approach
- Article search functionality
- Category/tag filtering
- Comment section integration
- Simple, efficient design

## 6. kentcdodds.com
**Layout:** Full-width modern grid, hero section with CTA
**Colors:** Light primary (#fff), blue accent (#0066cc), dark text
**Typography:** Optimized sans-serif, generous spacing
**Features:**
- Content hub (blog + courses + talks)
- Strong CTA sections
- Social proof (testimonials/metrics)
- Free/premium content split
- Email newsletter CTA
- Professional brand consistency

## 7. www.netlify.com/blog
**Layout:** Marketing-focused hero + card-based article grid
**Colors:** Bright teal accent (#00c7b7), dark navy primary
**Typography:** Branded sans-serif, clear hierarchy
**Features:**
- Author cards with images
- Category badges and filtering
- Featured/sticky articles
- Related articles widget
- Share buttons
- Sponsored content clearly marked
